# **App Name**: ChronoClass

## Core Features:

- Lead Management: Manage potential student inquiries and convert them into enrolled students using a user-friendly CRM interface.
- Student Management: Maintain detailed records of students, track their progress, and manage their enrollment status within the Firestore database.
- Class Scheduling: Create and manage class schedules, automatically sync them with Google Calendar, and notify enrolled students. All AI flow-managed.
- Smart Scheduling and Conflict Detection: AI tool that finds optimal times, and suggests available classrooms for class scheduling, and automatically identifies scheduling conflicts across timezones.
- Course Management: Create and organize course materials, define learning objectives, and manage course availability.
- Google Calendar and Meet Integration: Integrate with Google Calendar to sync schedules and generate Google Meet links for classes automatically. AI-driven integration flow.
- Assignment Tracking: Manage homework and assignments, track submissions, and provide feedback using Firestore to store data. The tracking will use AI to keep pace and provide feedback
- Attendance Tracking: Track student attendance using various methods, including manual input and automated attendance marking with AI-based face recognition (if video available) or geolocation.
- Personal Availability Management: Allow the super-user to define their availability across different time zones, to use for scheduling.
- Calendar Views: Provide Day, Week, Month views, and a 'Today' shortcut. Integrate with drag-and-drop rescheduling in the weekly view.
- Natural Language Class Creation: Enable class creation via natural language input. Process input like "Schedule a class on Monday at 2 PM for Advanced Math".
- Gemini Voice Assistant Integration: Allow management using voice commands through Gemini. Schedule, reschedule, and manage classes using voice commands from a phone.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a modern, intellectual feel.
- Background color: Light Gray (#F0F2F5), a very subtle tint of the primary hue, providing a clean and unobtrusive background.
- Accent color: Electric Purple (#8E44AD), a high-contrast accent to highlight interactive elements and important actions.
- Headline font: 'Space Grotesk' (sans-serif) for headlines, creating a tech-forward yet approachable aesthetic; body font: 'Inter' (sans-serif) for body text, chosen for its clarity and readability.
- Use modern, outline-style icons for a clean and consistent look. Icons should subtly reflect their function with minimal detail, providing easy recognition at a glance.
- Maintain a fluid, responsive layout that adapts seamlessly to different screen sizes. Use clear visual hierarchy to guide users through tasks. Employ white space effectively to avoid clutter and improve readability.
- Incorporate purposeful animations that enhance user experience without being distracting. Transitions should be smooth and responsive. Use subtle animations to highlight key actions.
